# 📊 Production Performance & Workforce Insights Dashboard

## Overview
This Power BI dashboard provides insights into production performance, runtime, breakdown time, and workforce efficiency. It enables monitoring of operational KPIs and comparison of actual efficiency against targets across production lines, groups, and sections.

## Key Metrics
- Total Production: 520.74K
- Total Runtime: 547.59K Hours
- Total Breakdown: 26.85K Hours
- Efficiency: 80.63%
- Performance Gap: -0.43%

## Features
- Dynamic filters by:
  - Year
  - Month
  - Line
  - Group
  - Section

- KPI cards for overall performance.
- Production and runtime analysis by line.
- Actual efficiency vs target comparison.
- Production distribution by group.
- Breakdown time analysis by section.
- Trend analysis of production, runtime, and breakdown over time.

## Dataset Columns

| Column Name | Description |
|-------------|-------------|
| Year | Production year |
| Calendar Day | Day of production |
| Week | Week number |
| Month | Month name |
| Total Time (Hours) | Total available hours |
| Runtime (NPT) (Hours) | Machine runtime hours |
| Breakdown Time (Hours) | Downtime due to breakdowns |
| Production Time (Hours) | Effective production hours |
| Maintenance Time (Hours) | Maintenance hours |
| Line | Production line |
| Performance Target | Target efficiency (%) |
| Maintenance Target | Target maintenance hours |
| Breakdown Target | Target breakdown hours |
